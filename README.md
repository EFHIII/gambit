# GamBit
A Discord bot for playing various games within the text environment including games like connect 4.

Currently supported games:
* Connect 4
* Othello
* Taboo
* Tic-Tac-Toe
* Texas Hold'em
* Mancala (Kalah)

Discord community link: [GamBit Discord](https://discord.gg/6gH8yqEsy8)
